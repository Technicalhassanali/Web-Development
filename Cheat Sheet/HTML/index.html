<meta> | Meta Tags

<meta> tags always go inside the <head> element, and are typically used to specify character set, page description, keywords, author of the document, and viewport settings.

<meter> </meter>

<meter> tag defines a scalar measurement within a known range, or a fractional value. This is also known as a gauge.
Examples: Disk usage, the relevance of a query result, etc.

<meter> tag should not be used to indicate progress (as in a progress bar).
For progress bars, use the <progress> tag.
Tips:Always add the <label> tag for best accessibility practices!.


Highattribute specifies the range where the gauge's value is considered to be a high value. The high attribute value must be less than the max attribute value, and it also must be greater than the low and min attribute values.

lowattribute specifies the range where the gauge's value is considered to be a low value. The low attribute value must be greater than the min attribute value, and it also must be less than the high and max attribute values.

minattribute specifies the lower bound of the gauge. The min attribute value must be less than the max attribute value. If unspecified, the default value is 0.

maxattribute specifies the upper bound of the gauge. The max attribute value must be greater than the min attribute value. If unspecified, the default value is 1.

optimum attribute specifies the range where the gauge's value is considered to be an optimal value.

value The required value attribute specifies the current, or "measured", value of the gauge. The value attribute must be between the min and max attribute values.

Simple Selector

select elements based on name, id, class

Element Selector

It is used to select an element based on the tag name

For example:

H2 {
          color: blue;
}

Id Selector

It is used to select an element with a given id

For example:

#first {
          color: white;
          background: black;
}

‘#’ is used to target by id

Class Selector

It is used to select an element with a given class

For example:

.red {
          background: red;
}

Combinator Selector

A combinator is something that explains the relationship between the selectors. A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator.
There are four different combinators in CSS:

  • Descendant selector (space)
  • child selector (>)
  • Adjacent sibling selector (+)
  • General sibling selector (~)

Descendant Selector

This selector is actually multiple selectors combined together, but separated by a space.Descendant selector is used to select all the elements which are child of the element (not a specific element).

Example:

In Blow Example You can see the Selector is highlight all element of p that present in div.

    
        div p {
            background-color: yellow;
          }

Child Selector

The child selector selects all elements that are the children of a specified element.

Example:

CSS Implicit & Explicit Grid

        

        
The Explicit Grid is the grid that you define with the grid-template-rows ,
            grid-template-columns , and grid-template-areas properties.

        
The Implicit Grid is automatically generated by the grid container whenever
            grid items are positioned outside of the explicit grid.

        
Properties :

        
    grid-auto-columns 
    grid-auto-rows 
    grid-auto-flow



        
grid-auto-rows when the new grid items come in the grid container then default
            set height used this property becuase the New
            grid items not define in the grid-template-rows , grid-template-columns.Therefore, we use this property.

        

        .grid-container{
            grid-auto-rows : 200px;
        }
    


        
grid-auto-columns  is used to set the width of new icomming grid item that are not define in grid-template-rows , grid-template-columns.Therefore, we use this property.

        

                .grid-container{
                    grid-auto-rows : 200px;
                    grid-auto-columns:.5fr; /*Half of width */
                }
            



        
grid-auto-flow when a new grid item is add then default it is add in the rows
            if we use this property
            then change the new grid items incomming and present in column wise.

        

    .grid-container{
        grid-auto-rows : 200px;
        grid-auto-flow:column;
    }
